ABOUT ME

AI 기능을 서비스 형태로 연결하는 백엔드 개발을 목표로 합니다. STT, 요약, RAG 기반 질의응답 파이프라인을 중심으로 데이터 처리, API 설계, 서비스 흐름 구성 경험을 보유하고 있습니다.

프로젝트에서는 모델 성능보다 데이터 구조와 입력 품질, 그리고 사용자 흐름을 우선해 설계합니다. 분류 프로젝트에서의 시행착오를 통해 문제를 데이터와 구조 관점에서 정의하는 방식의 중요성을 인식했습니다.

부트캠프 최종 팀 프로젝트 GENMINUTE에서 AI 회의록 생성 서비스를 개발하며, 음성 전사부터 요약·검색·질의응답까지 이어지는 파이프라인을 실제 서비스 흐름에 맞게 구현하고 마무리했습니다.
